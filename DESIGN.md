# 🎨 디자인 가이드 (적용 내역)

기준 문서: **`stock_dashboard_design.md`** — "예측 앱"이 아니라 차분한 **금융 분석 대시보드**.

## 적용된 방향
- **장난감 게이지 제거** → 종합 신호를 점수(0~100) + 가로 점수바 + 배지 카드로 표현
- **요약 카드 4개**: 현재가 · 전일 대비 · 종합 신호(보조지표) · 리스크
- **차트를 화면 중심**으로 풀폭 확대, 지표는 아래 카드 + 근거 테이블
- 예측성 문구 완화: "오를까 내릴까?" 제거 → "현재 지표 기반 모멘텀 요약"
- 노란 경고 박스 제거 → 작은 안내 캡션
- 이모지 최소화

## 토큰
| 용도 | 값 |
|------|----|
| 페이지 배경 | `#F6F8FB` |
| 카드 배경 | `#FFFFFF` |
| 텍스트 | `#111827` / 보조 `#6B7280` |
| 테두리 | `#E5E7EB` |
| 강조색(1개) | `#2563EB` |
| 상승 / 하락 / 중립 | `#E53935` / `#2563EB` / `#9CA3AF` (국내 관례) |
| 경고 | `#F59E0B` |

- 폰트: **Pretendard**(본문) + Inter 폴백
- 타이포: 페이지 타이틀 26 / 섹션 18~20 / 카드 숫자 22~28 (700)

## 구현 위치
- `.streamlit/config.toml` — 테마 팔레트
- `app.py` `CUSTOM_CSS` — Pretendard, 카드/배지/점수바/총평 박스 스타일
- `app.py` `card_html` · `risk_badges` · `indicator_mini_cards` — 카드 컴포넌트

## 남은 개선 아이디어 (문서 체크리스트 2·3차)
