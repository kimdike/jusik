# 자산 대시보드 디자인 가이드

## 1. 목표
현재 화면은 기능은 보이지만 기본 Streamlit/Claude 결과물 느낌이 강함.  
목표는 **투자자가 매일 열어보고 싶은 금융 서비스형 대시보드**로 개선하는 것.

핵심 방향:
- 좌측 메뉴 중심의 단순 앱 느낌 제거
- 카드 기반 정보 구조
- 차트 중심의 전문성 강화
- 매수/매도 추천처럼 보이는 표현은 완화
- 신호 해석은 “보조 지표”로 명확히 표현

## 2. 전체 톤앤매너

### 키워드
- Clean
- Financial
- Calm
- Data-first
- Professional
- Dashboard

### 피해야 할 느낌
- 기본 Streamlit 앱
- 과한 이모지
- 원색 경고 UI
- 장난감 같은 게이지
- 너무 큰 제목과 여백
- “오를까 내릴까?”처럼 예측 서비스로 보이는 문구

## 3. 레이아웃 구조

### 권장 화면 구조

```text
┌─────────────────────────────────────────────────────────────┐
│ Top Bar                                                     │
│ 자산 대시보드 / 검색 / 관심종목 / 기간 / 설정              │
├───────────────┬─────────────────────────────────────────────┤
│ Sidebar       │ Main Content                                │
│ Watchlist     │ ┌ Summary Cards ┐                           │
│ Market        │ ├ Price / Change / Signal / Risk ┤          │
│ Alerts        │                                             │
│ Settings      │ ┌ Main Chart ┐                              │
│               │ └───────────────────────────────┘           │
│               │ ┌ Indicator Cards / Tables ┐                │
└───────────────┴─────────────────────────────────────────────┘
```

### 개선 방향
현재처럼 좌측 라디오 메뉴가 화면을 크게 차지하지 않도록 한다.  
좌측은 **아이콘 + 짧은 메뉴명 + 관심종목 리스트** 중심으로 구성한다.

## 4. 정보 우선순위

### 1순위: 종목 요약
상단에 한 줄로 핵심 정보를 제공한다.

예시:
```text
삼성전자 005930 · KRX
353,500원  ▲ 2.4%  /  1년 수익률 +128.3%
```

### 2순위: 종합 신호
현재의 큰 게이지는 축소하고, 카드형으로 표현한다.

예시:
```text
종합 신호
상승 우세 86.5
단기 모멘텀 강함 · 과열 주의
```

### 3순위: 차트
차트가 화면의 중심이어야 한다.  
가격, 이동평균, 볼린저밴드, 저항/지지선은 한 영역에서 명확히 구분한다.

### 4순위: 세부 지표
RSI, MACD, ADX, 일목균형표 등은 아래 카드 또는 탭으로 분리한다.

## 5. 컬러 시스템

### 기본 컬러
```css
--bg-main: #F6F8FB;
--bg-card: #FFFFFF;
--text-primary: #111827;
--text-secondary: #6B7280;
--border-soft: #E5E7EB;
--accent-blue: #2563EB;
```

### 상승/하락 컬러
국내 주식 사용자 기준으로 상승은 빨강, 하락은 파랑을 사용한다.

```css
--up: #E53935;
--down: #2563EB;
--neutral: #9CA3AF;
--warning: #F59E0B;
```

### 배경 사용 원칙
- 메인 배경은 연한 회색
- 카드 배경은 흰색
- 강조색은 1개만 사용
- 노란 경고 박스는 최소화
- 신호는 원색 배경보다 작은 배지로 표현

## 7. 컴포넌트 디자인

## 7.1 상단 검색 영역
검색창과 종목 선택 드롭다운을 같은 줄에 두되, 카드 안에 넣는다.

```text
[ 종목명 또는 티커 검색                 ] [ 관심종목 선택 ▼ ] [ 분석 ]
```

검색 후 아래에 작은 메타 정보 표시:
```text
삼성전자 · 005930 · KRX · 데이터 기준 2026.06.22 15:30
```

---

## 7.2 요약 카드
상단에 4개 카드 배치.

```text
┌ 현재가 ┐ ┌ 등락률 ┐ ┌ 종합 신호 ┐ ┌ 리스크 ┐
```

예시:
```text
현재가
353,500원
전일 대비 +2.4%
```

```text
종합 신호
86.5 / 100
상승 우세
```

```text
리스크
과열 주의
RSI 70 근접
```

---

## 7.3 종합 신호 표현
큰 속도계 게이지는 제거하거나 축소한다.  
금융 대시보드에서는 아래 형태가 더 자연스럽다.

### 권장 1: Horizontal Score Bar
```text
종합 점수 86.5
[■■■■■■■■□□] 상승 우세
```

### 권장 2: Signal Distribution
```text
강한 상승 4
상승 3
중립 1
하락 1
강한 하락 1
```

### 권장 3: Badge
```text
[상승 우세] [모멘텀 강함] [과열 주의]
```

---

## 7.4 차트 영역
차트는 가장 큰 영역으로 배치한다.

권장 구성:
- 가격 캔들
- MA20 / MA60 / MA120
- 볼린저밴드
- 지지/저항선
- 거래량
- RSI
- MACD

차트 상단 컨트롤:
```text
1M 3M 6M 1Y 2Y 5Y  전체
[ ] 이동평균  [ ] 볼린저밴드  [ ] 지지저항  [ ] 거래량
```

차트 개선 원칙:
- 범례는 상단 좌측에 작게
- 선 색상은 최대 5개 이하
- 지지/저항선은 점선 + 낮은 채도
- 가격 캔들은 가장 눈에 띄게
- 보조지표는 접기/펼치기 가능

---

## 7.5 지표 상세 카드
차트 아래에 카드형 지표 설명을 배치한다.

```text
┌ RSI ┐ ┌ MACD ┐ ┌ 이동평균 ┐ ┌ 볼린저밴드 ┐
```

예시:
```text
RSI
64.2
과매수 구간 접근
```

```text
MACD
상승 크로스 유지
단기 모멘텀 양호
```

```text
이동평균
정배열
중장기 추세 우호적
```

## 9. 화면별 개선안

## 9.1 기술적 분석 화면

### 현재 문제
- 제목이 너무 크고 캐주얼함
- 경고 박스가 과함
- 게이지가 금융 서비스보다 장난감 느낌
- 차트와 요약 영역의 위계가 애매함
- 좌측 메뉴가 기본 앱 느낌을 줌

### 개선안
```text
[기술적 분석]
삼성전자 · 005930 · KRX

[현재가] [등락률] [종합 신호] [리스크]

[대형 차트]

[지표 상세 카드]

[신호 근거 테이블]
```

---

- 거래대금 상위

## 12. 최종 디자인 방향 요약

이 대시보드는 “예측 앱”이 아니라 **현재 시장 데이터를 정리해주는 분석 도구**로 보여야 한다.  
따라서 디자인은 화려한 예측 UI보다, 금융 서비스처럼 차분하고 신뢰감 있는 구조가 적합하다.

핵심은 다음 3가지다.

1. **차트 중심**으로 화면 위계 재정리
2. **종합 신호는 보조 카드**로 축소
3. **투자 조언처럼 보이는 문구는 완화**

결과적으로 사용자가 화면을 봤을 때 “AI가 찍어주는 앱”이 아니라  
“내가 판단하기 쉽게 정리해주는 투자 대시보드”로 느껴지도록 한다.
