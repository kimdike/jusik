# 📈 내 주식·자산 관리 대시보드

한국 주식 + 미국 주식 + 코인을 한 화면에서 관리하고, **여러 기술적 지표를 종합해
"지금 상승 신호가 우세한지 하락 신호가 우세한지"를 수치로** 보여주는 로컬 대시보드입니다.

> ⚠️ **중요:** 이 도구가 보여주는 "상승 우세 %"는 미래를 맞히는 확률이 아니라,
> 현재 여러 지표가 어느 방향 신호를 더 많이/강하게 내고 있는지를 점수화한 것입니다.
> 종목 추천이 아니며, 투자 판단의 **보조 지표**로만 사용하세요. 투자 책임은 본인에게 있습니다.

---

## 🚀 실행 방법

PowerShell에서 이 폴더(`C:\Users\KDH\Desktop\jusik`)로 이동 후:

```powershell
.venv\Scripts\streamlit run app.py
```

브라우저가 자동으로 열립니다 (안 열리면 http://localhost:8501 접속).
종료하려면 터미널에서 `Ctrl + C`.

## 🖥️ 화면 구성

| 메뉴 | 설명 |
|------|------|
| **기술적 분석** | **이름으로 종목 검색** → 캔들차트 + 8종 지표 + **상승/하락 5단계 신호 게이지** + **기업 지표(PER/PBR/시총/배당/52주)** + **지지·저항 가격대** + **관련 뉴스** (이 프로젝트의 핵심) |
| **시장 개요** | 코스피·코스닥·S&P500·나스닥·환율·VIX 등락률 + 코인 공포탐욕지수 + 시장 거시 뉴스 |
| **내 자산** | 보유 종목·수량·매입가 입력 → 현재가 기준 평가액·수익률 자동 계산 (미국주식은 원화 환산) |
| **워치리스트** | 이름 검색으로 관심종목 추가 + 현재가·종합 신호를 한 줄씩 모아 보기 |
| **알림 설정** | 신호 변화 / 목표가·손절가 도달 시 텔레그램 알림 (자세히는 `ALERTS.md`) |

### 종목 검색 팁
- **한국주식**: 한글 (삼성전자, 카카오) — KRX 상장사 전체
- **미국주식**: 영문(apple, tesla) 또는 인기종목 한글(애플·테슬라·엔비디아…)
- **코인**: 한글/영문 (비트코인, btc)

## 📊 종합에 쓰는 기술적 지표

이동평균(20/60/120 배열·골든/데드크로스), RSI, MACD, 볼린저밴드, 스토캐스틱,
**일목균형표(구름대·전환선/기준선)**, **피보나치 되돌림**, ADX(추세 강도), 추세선(선형회귀 기울기).

각 지표가 상승/하락/중립에 투표하고, 가중 합산해서 `상승 우세 %`를 계산합니다.

## 🔌 데이터 출처 (전부 무료, API 키 불필요)

- **주식 (한국/미국):** yfinance (Yahoo Finance)
  - 한국: 6자리 코드 (예: `005930`) → 코스피(.KS)/코스닥(.KQ) 자동 인식
  - 미국: 티커 그대로 (예: `AAPL`)
- **코인:** 업비트 공개 API (원화 마켓, 예: `BTC` → `KRW-BTC`)
- **종목 검색:** KRX 상장사 목록(한글명) + Yahoo 검색 + 업비트 마켓목록
- **뉴스:** Google News RSS (한/영)
- **시장 심리:** alternative.me 공포탐욕지수
- **펀더멘털:** yfinance 종목 정보

## 📁 구조

```
jusik/
├─ app.py              # Streamlit 대시보드 (UI, 5개 페이지)
├─ alerts_run.py       # 알림 러너 (1회/반복/테스트)
├─ src/
│  ├─ prices.py        # 가격·환율·펀더멘털 수집
│  ├─ indicators.py    # 기술적 지표 계산
│  ├─ signals.py       # 신호 종합 스코어(5단계) 엔진
│  ├─ levels.py        # 지지/저항 가격대
│  ├─ glossary.py      # 지표 설명·근거 해설
│  ├─ search.py        # 이름으로 종목 검색
│  ├─ news.py          # Google News RSS
│  ├─ market.py        # 시장 개요(지수·공포탐욕·거시뉴스)
│  ├─ notify.py        # 텔레그램 발송 (토큰은 코드/깃 미포함)
│  └─ alerts.py        # 알림 엔진 (신호변화·목표가·손절가)
├─ data/
│  ├─ portfolio.json   # 보유 종목      (대시보드에서 편집/저장)
│  ├─ watchlist.json   # 관심종목
│  └─ alerts.json      # 알림 설정
├─ requirements.txt
├─ README.md · DEPLOY.md · ALERTS.md
```

보유 종목/관심종목/알림은 대시보드 화면에서 추가·수정 후 **저장 버튼**을 누르면
위 JSON 파일에 기록됩니다.
